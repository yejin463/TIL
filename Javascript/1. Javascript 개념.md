## JS 개념

- 클래스 대신 proto를 사용한다.
- 자료형에 구애받지 않는다.
- 선언
  1. var:  스코프(범위)  함수,  너무 유연하게 사용되기 때문에 문제가 된다. **지양**
  2.  let: 스코프(범위)  중괄호, 재선언이 아닌 재정의
  3. const: 스코프(범위)  중괄호, 상수(바뀌지 않는 값)
